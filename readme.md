# tls-did-resolver

## Instalation

```
npm i
```

## Development

### Test

In tls-did-playground

```
npm run ganache
npm run truffle
```

```
npm run test
```

Please notice that repeating the test on the same instance of a ganache test network will lead to a multiple contract error.

### Build

Should be run before commit

```
npm run buils
```
